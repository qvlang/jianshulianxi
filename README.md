# 仿简书网站
```
该项目以简书为模板开发，实现了基本网站布局，手动实现轮播图。登录注册界面，以及前端登录鉴权。点击文字进入详情页面，实现详情页面基本布局展示
```
## 技术栈
+ React
+ React-Router-dom
+ Redux
+ immutable.js
